# Evaluate News NLP

## Overview: 
Evaluate News NLP is a web application that analyzes the sentiment of articles using Natural Language Processing (NLP) techniques. It allows users to submit article URLs, which are then analyzed to provide insights such as sentiment, confidence, agreement, and irony.

## Features 
- Setting up Webpack
- Sass styles
- Webpack Loaders and Plugins
- Creating layouts and page design
- Service workers
- Using APIs and creating requests to external URL

## Running The Project
### Install dependencies:
npm install

### Start the server:
npm start

### Build project:
npm run build-prod

## Getting Started

To get started with the Evaluate News NLP application, follow these steps:

1. Clone the repository to your local machine using the following command:

   ```
   gh repo clone Raqiya-AlMoqbali/evaluate-news-nlp
   ```
2. Set up the necessary environment and dependencies as specified in the project's documentation or README file.
3. Run the application locally using the provided scripts or commands.
4. Input the URL of a news article you wish to analyze into the application's input field.
5. Click the "Submit" button to initiate the sentiment analysis process.
6. View the sentiment analysis results displayed on the user interface.
