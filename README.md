# Evaluate News NLP

## Overview: 
Evaluate News NLP is a web application that analyzes the sentiment of articles using Natural Language Processing (NLP) techniques. It allows users to submit article URLs, which are then analyzed to provide insights such as sentiment, confidence, agreement, and irony.

## Features 
- Setting up Webpack
- Sass styles
- Webpack Loaders and Plugins
- Creating layouts and page design
- Service workers
- Using APIs and creating requests to external urls

## Running The Project
### Install dependencies:
npm install

### Start the server:
npm start

### Build project:
npm run build-prod

## Usage
- Enter the URL of the article you want to analyze in the provided input field.
- Click the "Submit" button to trigger the sentiment analysis.
- View the sentiment analysis results displayed on the page.
